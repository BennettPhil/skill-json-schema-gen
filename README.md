# json-schema-gen

Infer JSON Schema from sample JSON data with type detection and pattern recognition.

## Quick Start

```bash
./scripts/run.sh sample.json
./scripts/run.sh a.json b.json --merge
echo '{"name": "Alice"}' | ./scripts/run.sh --stdin
```
